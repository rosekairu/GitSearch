Git-Search (GitHub Search) is a website that enables the search of GitHub users and their repositories.

To get the full experience of the page, follow the following steps:

Click on the buttons on the navigation bar.

The Search-User navigates to the home page where you can view the webpage developer's github profile and also get to search other GitHub User Profiles and Repository.

The Search-Repo button navigates to the home page where you can get the preview of All GitHub Repositories.

The About button navigates to this page which gives more information about this webapp

The Home button navigates to the very begining.
